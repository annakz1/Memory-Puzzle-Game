Memory Game with
Auto-Detection

Welcome to the Memory Game with Auto detection! In this game you will upload 12 images containing 6 pairs, each pair is an image of the same object/scene in different angle or perspective.
After uploading the images, press RUN THE ALGORITHM and the matching algoritm will run and match each pair.
Now, you can START THE GAME! It will pop as a new program. At first you will see all the images will numbers on them- which represent each pair the Algorithm matched. After your first click, you can start playing :) Enjoy!

Note: the supported file formats are: png, jpg, jpeg.
